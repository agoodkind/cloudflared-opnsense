Cloudflared Package Repository

FreeBSD package repository for cloudflared OPNsense plugin.

Configuration

Add to /usr/local/etc/pkg/repos/cloudflared.conf:

cloudflared: {
  url: "https://cloudflared-opnsense.pkg.goodkind.io",
  enabled: yes,
  priority: 10
}

Installation

pkg update
pkg install os-cloudflared
